## 📝 NOTATKI TECHNICZNE

### Baza danych
- Tabela: `wp_woo_quick_ratings`
- Indeksy: product_id, user_id, ip_address, product_user, product_ip
- Pola: id, product_id, rating, user_id, ip_address, user_agent, created_at, updated_at

### Cache
- Klucze transients: `ihumbak_wrs_stats_{product_id}`
- TTL: 1 godzina (HOUR_IN_SECONDS)
- Automatyczne czyszczenie po update

### REST API
- Namespace: `woo-quick-ratings/v1`
- Endpoints:
  - POST `/rate` - dodaj ocenę
  - GET `/stats/{product_id}` - pobierz statystyki

### Hooki
- `woocommerce_single_product_summary` - wyświetlenie widżetu
- `woocommerce_product_get_average_rating` - modyfikacja średniej
- `woocommerce_product_get_rating_count` - modyfikacja liczby ocen
- `wp_enqueue_scripts` - ładowanie CSS/JS
- `rest_api_init` - rejestracja REST API
- `admin_menu` - menu w adminie
